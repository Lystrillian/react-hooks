NOT logical operator

## Usage

```tsx
import { logicNot } from '#';
import { whenever } from '#';

const a = ref(true);

whenever(logicNot(a), () => {
  console.log('a is now falsy!');
});
```

## Type Declarations

```typescript
/**
 * Uses a logical `NOT` operator with multiple values or functions.
 */

export { logicNot as not };
```
