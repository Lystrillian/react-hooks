The `useArrayFindIndex` hook provides a memoized way to find the index of the first element in an array that satisfies a given predicate function.

## Usage

Use with an array of multiple refs:

```tsx
import { useArrayFindIndex } from '#';

const Component = () => {
  const users = [
    { id: 1, name: 'Alice' },
    { id: 2, name: 'Bob' },
  ];

  const findUserIndexById = user => user.id === 2;
  const userIndex = useArrayFindIndex(users, findUserIndexById);

  return <div>User Index: {userIndex}</div>; // Output: 1
};
```

## Type Declarations

```typescript
export function useArrayFindIndex<T>(
  list: T[],
  predicate: UseArrayFindIndexPredicate<T>
): number;

export type UseArrayFindIndexPredicate<T> = (
  element: T,
  index: number,
  array: T[]
) => boolean;
```
