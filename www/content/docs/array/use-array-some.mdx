Array.some.

## Usage

```tsx
import { useArraySome } from '#';

const numbers = [1, 2, 3, 4, 5];
const hasEvenNumber = useArraySome(numbers, num => num % 2 === 0);
// hasEvenNumber: true
```

Use with a custom predicate function:

```tsx
import { useArraySome } from '#';

const users = [
  { id: 1, active: false },
  { id: 2, active: true },
  { id: 3, active: false },
];

const hasActiveUser = useArraySome(users, user => user.active);
// hasActiveUser: true
```
