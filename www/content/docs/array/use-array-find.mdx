The `useArrayFind` hook offers a way to search for the first element within an array that meets a specific condition defined by a predicate function.

## Usage

```tsx
import { useArrayFind } from '#';

const numbers = [1, 2, 3, 4, 5];
const firstEvenNumber = useArrayFind(numbers, num => num % 2 === 0);
// firstEvenNumber: 2
```

Use with a custom predicate function:

```tsx
import { useArrayFind } from '#';

const users = [
  { id: 1, name: 'John', active: false },
  { id: 2, name: 'Jane', active: true },
  { id: 3, name: 'Bob', active: false },
];

const firstActiveUser = useArrayFind(users, user => user.active);
// firstActiveUser: { id: 2, name: 'Jane', active: true }
```

## Type Declarations

```typescript
export function useArrayFind<T>(
  list: T[],
  predicate: UseArrayFindPredicate<T>
): T | undefined;

export type UseArrayFindPredicate<T> = (
  element: T,
  index: number,
  array: T[]
) => boolean;
```
