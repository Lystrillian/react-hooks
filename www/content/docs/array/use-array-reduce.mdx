Array.reduce.

## Usage

```tsx
import { useArrayReduce } from '#';

const numbers = [1, 2, 3, 4, 5];
const sum = useArrayReduce(numbers, (acc, num) => acc + num, 0);
// sum: 15
```

Use with a custom reducer function:

```tsx
import { useArrayReduce } from 'react-hook-port';

const users = [
  { id: 1, name: 'John', age: 30 },
  { id: 2, name: 'Jane', age: 25 },
  { id: 3, name: 'Bob', age: 40 },
];

const totalAge = useArrayReduce(users, (acc, user) => acc + user.age, 0);
// totalAge: 95
```
