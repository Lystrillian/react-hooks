Array.join.

## Usage

```tsx
import { useArrayJoin } from '#';

const fruits = ['Apple', 'Banana', 'Orange'];
const joinedFruits = useArrayJoin(fruits);
// joinedFruits: "Apple,Banana,Orange"
```

Use with a custom separator:

```tsx
import { useArrayJoin } from '#';

const words = ['Hello', 'World', 'React'];
const joinedWords = useArrayJoin(words, ' ');
// joinedWords: "Hello World React"
```

## Type Declarations

```typescript
export declare function useArrayJoin<T>(list: T[], separator?: string): string;
```
