The `useArrayFindLast` hook offers an optimized approach to search for the _last_ element within an array that satisfies a provided predicate function.

## Usage

```tsx
import { useArrayFindLast } from '#';

const numbers = [1, 2, 3, 4, 5];
const lastEvenNumber = useArrayFindLast(numbers, num => num % 2 === 0);
// lastEvenNumber: 4
```

Use with a custom function:

```tsx
import { useArrayFindLast } from '#';

const users = [
  { id: 1, name: 'John', active: true },
  { id: 2, name: 'Jane', active: false },
  { id: 3, name: 'Bob', active: true },
];

const lastActiveUser = useArrayFindLast(users, user => user.active);
// lastActiveUser: { id: 3, name: 'Bob', active: true }
```

## Type Declarations

```typescript
export function useArrayFindLast<T>(
  list: T[],
  predicate: UseArrayFindLastPredicate<T>
): T | undefined;

export type UseArrayFindLastPredicate<T> = (
  element: T,
  index: number,
  array: T[]
) => boolean;
```
