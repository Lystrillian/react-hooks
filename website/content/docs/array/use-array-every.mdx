`useArrayEvery` is a React hook that checks if a given predicate function returns a truthy value for every element in an array. It mimics the behavior of JavaScript's `Array.every` method.

## Usage

```tsx
import { useArrayEvery } from '#';
import * as React from 'react';

const Component = () => {
  const [list, setList] = React.useState([2, 4, 6, 8]);

  // Check if all numbers are even
  const allEven = useArrayEvery(list, num => num % 2 === 0);

  return (
    <div>
      <p>All numbers are even: {JSON.stringify(allEven)}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { useArrayEvery } from '#';
import * as React from 'react';

const Component = () => {
  const [list, setList] = React.useState([
    { name: 'Alice', age: 25 },
    { name: 'Bob', age: 30 },
    { name: 'Charlie', age: 35 },
  ]);

  // Check if all people are at least 18 years old
  const allAdults = useArrayEvery(list, person => person.age >= 18);

  return (
    <div>
      <p>All people are adults: {JSON.stringify(allAdults)}</p>
    </div>
  );
};

export default Component;
```

## Type Declarations

```typescript
export declare function useArrayEvery<T>(
  list: T[],
  predicate: UseArrayEveryPredicate<T>
): boolean;

export declare type UseArrayEveryPredicate<T> = (
  element: T,
  index: number,
  array: T[]
) => unknown;
```

## Reference

| Property    | Type                        | Default | Description                      |
| ----------- | --------------------------- | ------- | -------------------------------- |
| `list`      | `T[]`                       | -       | The array to check.              |
| `predicate` | `UseArrayEveryPredicate<T>` | -       | A function to test each element. |

### Returns

| Type      | Description                                                                           |
| --------- | ------------------------------------------------------------------------------------- |
| `boolean` | A `boolean` value indicating whether all elements in the array satisfy the predicate. |

## Common Pitfals

### Invalid Predicate Function

Ensure thhat `predicate` is a valid function

Example mistake:

```tsx
const result = useArrayEvery([1, 2, 3], 'not-a-function');
```

**Solution:** Pass a valie function to `predicate`

```tsx
const result = useArrayEvery([1, 2, 3], num => num > 0);
```

### Invalid Array

Ensure that `list` is a valid array.

Example mistake:

```tsx
const result = useArrayEvery('not-an-array', num => num > 0);
```

**Solution:** Pass a valie array to `list`.

```tsx
const result = useArrayEvery([1, 2, 3], num => num > 0);
```
