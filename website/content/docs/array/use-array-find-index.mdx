A hook providing a memoized version of the `Array.findIndex` method for React, finding the index of the first element that passes the specified predicate function.

## Usage

Basic Example:

```tsx
import { useArrayFindIndex } from '#';
import { useState } from 'react';

const Component = () => {
  const [users, setUsers] = useState([
    { id: 1, name: 'Alice' },
    { id: 2, name: 'Bob' },
    { id: 3, name: 'Charlie' },
  ]);

  const findUserIndexById = user => user.id === 2;

  const userIndex = useArrayFindIndex(users, findUserIndexById);

  return (
    <div>
      <p>User Index: {userIndex}</p>
    </div>
  );
};
```

Advanced Example:

```tsx
import { useArrayFindIndex } from '#';
import { useState } from 'react';

const Component = () => {
  const [books, setBooks] = useState([
    { title: 'Book A', author: 'Author A' },
    { title: 'Book B', author: 'Author B' },
  ]);

  const findBookByTitle = book => book.title === 'Book B';

  const bookIndex = useArrayFindIndex(books, findBookByTitle);

  return (
    <div>
      <p>Book Index: {bookIndex}</p>
    </div>
  );
};
```

## Type Declarations

```typescript
export type UseArrayFindIndexPredicate<T> = (
  element: T,
  index: number,
  array: T[]
) => boolean;

export function useArrayFindIndex<T>(
  list: T[],
  predicate: UseArrayFindIndexPredicate<T>
): number
```

## API Details

### Parameters

`list` (`T[]`): The array to search.

`predicate` (`UseArrayFindIndexPredicate<T>`): A function to test each element in the array, returning `true` for the element to include.

### Returns

`number`: The index of the first element in the array that passes the test. Otherwise, `-1`.

## Common Pitfals

### Providing a predicate that isn't a function

Ensure thhat `predicate` is a valid function

```tsx
const books = [{ title: 'Book A' }, { title: 'Book B' }];
const invalidPredicate = 'findBookByTitle';

const bookIndex = useArrayFindIndex(books, invalidPredicate); // Error
```
