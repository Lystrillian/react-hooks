a hook providing a memoized version of the `Array.join` method for React, joining all elements of an array into a string separated by the specified separator.

## Usage

```tsx
import useArrayJoin from '#';
import * as React from 'react';

const Component = () => {
  const [words, setWords] = React.useState(['hello', 'world']);

  const joinedWords = useArrayJoin(words, ' ');

  return (
    <div>
      <p>Joined Words: {joinedWords}</p>
    </div>
  );
};

export default Component;
```

```tsx
import useArrayJoin from '#';
import * as React from 'react';

const Component = () => {
  const [numbers, setNumbers] = React.useState([1, 2, 3]);

  const joinedNumbers = useArrayJoin(numbers, '-');

  return (
    <div>
      <p>Joined Numbers: {joinedNumbers}</p>
    </div>
  );
};

export default Component;
```

## Type Declarations

````typescript
export declare function useArrayJoin<T>(list: T[], separator?: string): string```
````

## Reference

| Property    | Type     | Default | Description                                                                               |
| ----------- | -------- | ------- | ----------------------------------------------------------------------------------------- |
| `list`      | `T[]`    | -       | The primary array to join.                                                                |
| `separator` | `string` | `,`     | (_Optional_) A string to separate each pair of adjacent elements in the resulting string. |

### Returns

| Type     | Description                                                                                    |
| -------- | ---------------------------------------------------------------------------------------------- |
| `string` | A string with all array elements joined. If `list.length` is `0`, an empty string is returned. |

## Common Pitfals

### Providing an invalid separator

```tsx
const words = ['hello', 'world'];
const invalidSeparator = '!';

const joinedWords = useArrayJoin(words, invalidSeparator); // Error
```
