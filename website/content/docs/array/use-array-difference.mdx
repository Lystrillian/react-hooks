`useArrayDifference` calculates the difference between two arrays. It returns a new array containing elements from the first array that are not present in the second array.

## Usage

```tsx
import { useArrayDifference } from '#';
import * as React from 'react';

const Component = () => {
  const [list1, setList1] = React.useState([0, 1, 2, 3, 4, 5]);
  const [list2, setList2] = React.useState([4, 5, 6]);

  const difference = useArrayDifference(list1, list2);

  return (
    <div>
      <p>Difference: {JSON.stringify(difference)}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { useArrayDifference } from '#';
import * as React from 'react';

const Component = () => {
  const [list1, setList1] = React.useState([{ id: 1 }, { id: 2 }, { id: 3 }]);
  const [list2, setList2] = React.useState([{ id: 2 }, { id: 4 }]);

  // Custom comparison function based on the 'id' property
  const difference = useArrayDifference(list1, list2, (a, b) => a.id === b.id);

  return (
    <div>
      <p>Difference: {JSON.stringify(difference)}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { useArrayDifference } from '#';
import * as React from 'react';

const Component = () => {
  const [list1, setList1] = React.useState([
    { name: 'Alice' },
    { name: 'Bob' },
  ]);
  const [list2, setList2] = React.useState([
    { name: 'Bob' },
    { name: 'Charlie' },
  ]);

  // Compare based on the 'name' property
  const difference = useArrayDifference(list1, list2, 'name');

  return (
    <div>
      <p>
        Difference showing elements from list1 that are not present in list2:{' '}
        {JSON.stringify(difference)}
      </p>
    </div>
  );
};

export default Component;
```

## Type Declarations

```typescript
export declare function useArrayDifference<T>(
  list: T[],
  values: T[],
  keyOrCompareFn?: keyof T | UseArrayDifferenceComparatorFn<T>
): T[];

export declare type UseArrayDifferenceComparatorFn<T> = (
  element: T,
  value: T
) => boolean;

export declare type UseArrayDifferenceKey<T> =
  | keyof T
  | UseArrayDifferenceComparatorFn<T>;
```

## Reference

#### useArrayDifference

| Property         | Type                                           | Default | Description                                                          |
| ---------------- | ---------------------------------------------- | ------- | -------------------------------------------------------------------- |
| `list`           | `T[]`                                          | -       | The primary array.                                                   |
| `values`         | `T[]`                                          | -       | The array to compare against.                                        |
| `keyOrCompareFn` | `keyof T \| UseArrayDifferenceComparatorFn<T>` | -       | (_Optional_) The key to use for comparison or a comparator function. |

#### UseArrayDifferenceKey

| Property                         | Type                                | Description                                                                       |
| -------------------------------- | ----------------------------------- | --------------------------------------------------------------------------------- |
| `element`                        | `T`                                 | An element frmo the `list` array.                                                 |
| `value`                          | `T`                                 | An element frmo the `values` array..                                              |
| `UseArrayDifferenceComparatorFn` | `(element: T, value: T) => boolean` | Comparator function that returns `true` if the two elements are considered equal. |

### Returns

| Type  | Description                                                                                       |
| ----- | ------------------------------------------------------------------------------------------------- |
| `T[]` | Returns a new array containing elements from the `list` that are not present in the values array. |

## Common Pitfals

### Incorrect Key Usage:

If using a key for comparison, ensure that the key exists in all objects.

```ts
const difference = useArrayDifference([{ id: 1 }], [{ name: 'Alice' }], 'id');
```

**Solution:** Ensure both arrays have the key `id`.

### Inconsistent Data Types:

Ensure both arrays contain comparable data types, especially when using direct comparison.
