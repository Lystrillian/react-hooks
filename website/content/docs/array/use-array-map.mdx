A hook providing a memoized version of the `Array.map` method for React, creating a new array populated with the results of calling a provided function on every element in the given array. A

## Usage

```tsx
import { useArrayMap } from '#';
import { useState } from 'react';

const Component = () => {
  const [numbers, setNumbers] = useState([1, 2, 3, 4]);

  const squaredNumbers = useArrayMap(numbers, number => number * number);

  return (
    <div>
      <p>Squared Numbers: {squaredNumbers.join(', ')}</p>
    </div>
  );
};
```

```tsx
import { useArrayMap } from '#';
import { useState } from 'react';

const Component = () => {
  const [users, setUsers] = useState([{ name: 'Alice' }, { name: 'Bob' }]);

  const userNames = useArrayMap(users, user => user.name);

  return (
    <div>
      <p>User Names: {userNames.join(', ')}</p>
    </div>
  );
};
```

## Type Declarations

````typescript
export function useArrayMap<T, U = T>(
  list: T[],
  callback: (element: T, index: number, array: T[]) => U
): U[]```
````

## API Details

### Parameters

| Property    | Type                           | Default | Description                                                                                                      |
| ----------- | ------------------------------ | ------- | ---------------------------------------------------------------------------------------------------------------- |
| `list`      | `T[]`                          | -       | The primary array to map over.                                                                                    |
| `callback`  | `(element: T, index: number, array: T[]) => U` | -       | A function that is called for every element of the given list. Each time callback executes, the returned value is added to the new array. |

### Returns

| Property  | Type      | Default | Description                                                     |
| --------- | --------- | ------- | --------------------------------------------------------------- |
| `U[]`     | `U[]`     | -       | A new array with each element being the result of the callback function. |

## Common Pitfals

### Providing a callback that isn't a function

```tsx
const numbers = [1, 2, 3, 4];
const invalidCallback = 'number * number';

const squaredNumbers = useArrayMap(numbers, invalidCallback); // Error
```
