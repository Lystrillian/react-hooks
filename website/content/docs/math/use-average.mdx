`useAverage` is a custom React hook that calculates the average of the provided values.

## Usage

```tsx
import { useAverage } from '#';
import * as React from 'react';

const Component = () => {
  const avg = useAverage(1, 2, 3, 4); // Returns `2.5`
  const avgArray = useAverage([1, 2, 3, 4]); // Returns `2.5`

  return (
    <div>
      <p>Average: {avg}</p>
      <p>Average (Array): {avgArray}</p>
    </div>
  );
};

export default Component;
```

## Common Pitfalls

### Mixing Individual Values and Arrays

Ensure that all individual values or a single array are passed as arguments consistently.

```tsx
const avg = useAverage(1, [2, 3], 4); // Incorrect, should be consistent
```

### Non-Numeric Values

Ensure that all provided values are numeric.

```tsx
const avg = useAverage(1, 2, 'three', 4); // Incorrect, "three" is not numeric
```
