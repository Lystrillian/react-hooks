`useCeil` is a React hook that returns the smallest integer greater than or equal to a given number using `Math.ceil`.

## Usage

```tsx
import { useCeil } from '#';
import * as React from 'react';

const Component = () => {
  const ceilValue = useCeil(4.2); // Returns `5`
  const ceilValueFromFn = useCeil(() => 4.7); // Returns `5`

  return (
    <div>
      <p>Ceiling Value: {ceilValue}</p>
      <p>Ceiling Value (Function): {ceilValueFromFn}</p>
    </div>
  );
};

export default Component;
```

## Type Declarations

```ts
export declare type Resolvable<T> = T | (() => T);

export declare function resolveValue<T>(value: Resolvable<T>): T;

export declare function useCeil(value: Resolvable<number>): number;
```

## Reference

| Parameter | Type                 | Description              |
| --------- | -------------------- | ------------------------ |
| `value`   | `Resolvable<number>` | The value to be clamped. |

### Returns

| Type     | Description        |
| -------- | ------------------ |
| `number` | The clamped value. |

## Common Pitfalls

### Incorrect Data Type

Ensure that the value passed to `useCeil` is a number or a function returning a number.

```tsx
const ceilValue = useCeil("4.2"); // Incorrect, should be a number
```

### Function Not Returning a Number

When using a function as the input, make sure the function returns a valid number.

```tsx
const ceilValue = useCeil(() => "4.7"); // Incorrect, should return a number
```
