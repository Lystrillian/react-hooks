`useMax` is a hook that returns the maximum value from the given arguments or an array of values.

## Usage

```tsx
import { useMax } from '#';
import * as React from 'react';

const Component = () => {
  const value1 = 5;
  const value2 = 10;
  const value3 = 3;

  const maxValue = useMax(value1, value2, value3);

  return (
    <div>
      <p>Maximum value: {maxValue}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { useMax } from '#';
import * as React from 'react';

const Component = () => {
  const values = [8, 2, 6, 4];

  const maxValueFromArray = useMax(values);

  return (
    <div>
      <p>Maximum value from array: {maxValueFromArray}</p>
    </div>
  );
};

export default Component;
```

### Returns

| Type     | Description                                                                                                                                                                     |
| :------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `number` | The maximum value from the given arguments or array of values. If any of the provided values or functions are functions, they are called and their return values are evaluated. |

## Common Pitfalls

### Passing Non-Numeric Values

Ensure that the values passed to `useMax` are numeric or can be coerced to numbers. Passing non-numeric values may lead to unexpected behavior.

```tsx
const maxValue = useMax(1, '2', true); // Incorrect usage
```

**Solution**: Pass numeric values or values that can be coerced to numbers.

```tsx
const maxValue = useMax(1, 2, 3);
// or
const maxValue = useMax([1, 2, 3]);
```
