`useSum` is a custom React hook that takes an array of numbers and calculates the sum of those numbers. It returns the calculated sum as a number.

## Usage

```tsx
import { useSum } from '#';
import * as React from 'react';

const Component = () => {
  const numbers = [1, 2, 3, 4, 5];
  const sum = useSum(numbers);

  return (
    <div>
      <p>Sum: {sum}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { useSum } from '#';
import * as React from 'react';

const Component = () => {
  const [numbers, setNumbers] = React.useState([10, 20, 30]);
  const sum = useSum(numbers);

  const handleAddNumber = () => {
    setNumbers([...numbers, 40]);
  };

  return (
    <div>
      <p>Sum: {sum}</p>
      <button onClick={handleAddNumber}>Add Number</button>
    </div>
  );
};

export default Component;
```

## Common Pitfalls

### Passing Non-Array Input

Ensure that the input passed to `useSum` is an array. Passing a non-array value will result in an error message being logged to the console.

```tsx
const sum = useSum(123); // Incorrect usage
```

**Solution**: Pass an array of numbers to useSum.

```tsx
const numbers = [1, 2, 3];
const sum = useSum(numbers);
```

### Passing Array with Non-Number Elements

An empty array will return a sum of `0`.

Make sure that the input array contains only numbers. If the array contains non-number elements, an error message will be logged to the console.

```tsx
const numbers = [1, '2', 3]; // Incorrect usage
const sum = useSum(numbers);
```

**Solution**: Ensure that the input array contains only numbers.

```tsx
const numbers = [1, 2, 3];
const sum = useSum(numbers);
```
