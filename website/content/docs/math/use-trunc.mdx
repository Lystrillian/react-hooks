`useTrunc` is a custom React hook that provides a truncated number and a function to update it by removing the fractional part of a given number. It uses the `Math.trunc()` function internally to perform the truncation operation.

## Usage

```tsx
import { useTrunc } from '#';
import * as React from 'react';

const Component = () => {
  const [truncatedNumber, truncateNumber] = useTrunc();

  return (
    <div>
      <p>Truncated number: {truncatedNumber}</p>
      <button onClick={() => truncateNumber(3.14)}>Truncate 3.14</button>
      <button onClick={() => truncateNumber(2.78)}>Truncate 2.78</button>
    </div>
  );
};

export default Component;
```

```tsx
import { useTrunc } from '#';
import * as React from 'react';

const Component = () => {
  const [truncatedNumber, truncateNumber] = useTrunc(5);

```

## Type Declarations

```ts
export declare function useTrunc(
  initialValue?: number
): [number, TruncateNumberFn];

export declare type TruncateNumberFn = (number: number) => void;
```

### Reference

#### useTrunc

| Property       | Type     | Default | Description                                                            |
| -------------- | -------- | ------- | ---------------------------------------------------------------------- |
| `initialValue` | `number` | `0`     | (_Optional_) The initial value of the truncated number. Defaults to 0. |

### Returns

| Type                         | Description                                                                   |
| ---------------------------- | ----------------------------------------------------------------------------- |
| `[number, TruncateNumberFn]` | An array containing the current truncated number and a function to update it. |

#### TruncateNumberFn

| Property | Type     | Description                 |
| -------- | -------- | --------------------------- |
| `number` | `number` | The number to be truncated. |

## Common Pitfalls

### Passing Non-Number Values

Ensure that the value passed to the `truncateNumber` function is a number. Passing non-number values will result in `NaN` (Not-a-Number) and an error message will be logged to the console.

```tsx
truncateNumber('3.14'); // Incorrect usage
```

**Solution**: Pass a number value to the `truncateNumber` function.

```tsx
truncateNumber(3.14);
```

### Forgetting to Destructure the Hook Return Value

Make sure to destructure the return value of the `useTrunc` hook into a truncated number and a function to update it.

```tsx
const truncatedNumber = useTrunc(); // Incorrect usage
```

**Solution**: Destructure the return value of the `useTrunc` hook.

```tsx
const [truncatedNumber, truncateNumber] = useTrunc();
```
