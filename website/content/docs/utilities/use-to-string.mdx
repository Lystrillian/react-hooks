`useToString` is a React hook that converts a value or a function that returns a value into a string representation.

## Usage

```tsx
import { useToString } from '#';
import * as React from 'react';

const Component = () => {
  const value = 42;
  const stringValue = useToString(value);

  return (
    <div>
      <p>String Value: {stringValue}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { useToString } from '#';
import * as React from 'react';

const Component = () => {
  const getRandomNumber = () => Math.random();
  const stringValue = useToString(getRandomNumber);

  return (
    <div>
      <p>String Value: {stringValue}</p>
    </div>
  );
};

export default Component;
```

## Type Declarations

```typescript
export declare function useToString<T>(value: Resolvable<T>): string;

export declare type Resolvable<T> = T | (() => T);
```

## Reference

#### useToString

| Property | Type            | Default | Description                                              |
| -------- | --------------- | ------- | -------------------------------------------------------- |
| `value`  | `Resolvable<T>` | -       | The value or a function that returns a value to convert. |

### Returns

| Type     | Description                             |
| -------- | --------------------------------------- |
| `string` | The string representation of the value. |

## Common Pitfalls

### Passing Incorrect Value Type

Ensure that the value passed to `useToString` is of a type that can be converted to a string.

```tsx
const value = {};
const stringValue = useToString(value); // Outputs: "[object Object]"
```

**Solution**: Pass a value that has a meaningful string representation or provide a custom function to resolve the value.
