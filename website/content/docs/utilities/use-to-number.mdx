`useToNumber` is a hook that converts a string or a number to a numeric value using specified options. It provides flexibility in choosing the conversion method (`parseFloat` or `parseInt`) and handling NaN values.

## Usage

```tsx
import { useToNumber } from '#';
import * as React from 'react';

const Component = () => {
  const stringValue = '42.42';
  const number = useToNumber(stringValue);

  return (
    <div>
      <p>Number: {number}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { useToNumber } from '#';
import * as React from 'react';

const Component = () => {
  const stringValue = '42';
  const number = useToNumber(stringValue, { method: 'parseInt', radix: 10 });

  return (
    <div>
      <p>Number: {number}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { useToNumber } from '#';
import * as React from 'react';

const Component = () => {
  const invalidValue = 'invalid';
  const number = useToNumber(invalidValue, { nanToZero: true });

  return (
    <div>
      <p>Number: {number}</p>
    </div>
  );
};

export default Component;
```

## Type Declarations

```typescript
export declare function useToNumber(
  value: string | number,
  options?: UseToNumberOptions
): number;

export interface UseToNumberOptions {
  method?: 'parseFloat' | 'parseInt';
  radix?: number;
  nanToZero?: boolean;
}
```

## Reference

#### useToNumber

| Property  | Type                 | Default  | Description                              |
| --------- | -------------------- | -------- | ---------------------------------------- |
| `value`   | `string`             | `number` | - The value to be converted to a number. |
| `options` | `UseToNumberOptions` | `{}`     | (_Optional_) Conversion options.         |

#### UseToNumberOptions

| Property    | Type         | Default      | Description                                                               |
| ----------- | ------------ | ------------ | ------------------------------------------------------------------------- |
| `method`    | `parseFloat` | `parseFloat` | (_Optional_) Specifies the method to convert the value.                   |
| `radix`     | `number`     | `10`         | (_Optional_) The radix to use when `method` is 'parseInt'.                |
| `nanToZero` | `boolean`    | `false`      | (_Optional_) If true, returns 0 instead of NaN when the conversion fails. |

### Returns

| Type     | Description                                    |
| -------- | ---------------------------------------------- |
| `number` | The numeric representation of the input value. |
