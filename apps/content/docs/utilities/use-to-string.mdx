`useToString` is a React hook that converts a value or a function that returns a value into a string representation.

## Installation

```bash
npx changeelog/reactuse@latest add useToString
```

## Usage

```tsx
import { useToString } from '#';
import * as React from 'react';

const Component = () => {
  const getRandomNumber = () => Math.random();
  const stringValue = useToString(getRandomNumber);

  return (
    <div>
      <p>String Value: {stringValue}</p>
    </div>
  );
};

export default Component;
```

## Type Declarations

```typescript
export declare function useToString<T>(value: Resolvable<T>): string;

export declare type Resolvable<T> = T | (() => T);
```
