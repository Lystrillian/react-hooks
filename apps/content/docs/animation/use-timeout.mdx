---
title: useTimeout
description: Update value after a given time with controls.
---

## Installation

```bash
npx changeelog/reactuse@latest add useTimeout
```

## Usage

```tsx
import { useTimeout } from '#';

const Component = () => {
  const { ready, start, stop } = useTimeout(1000, { controls: true });

  return (
    <div>
      <p>{ready ? 'Timeout completed' : 'Waiting...'}</p>
      <button onClick={start}>Start</button>
      <button onClick={stop}>Stop</button>
    </div>
  );
};
```

## Reference

### useTimeout

`useTimeout` is a hook that creates a timeout and optionally returns controls for managing it.

### UseTimeoutOptions

An interface defining optional configuration for the timeout:

<TypeTable
  type={{
    immediate: {
      description: 'Start the timer immediately after calling this function.',
      type: 'boolean',
      default: 'true',
    },
    controls: {
      description: 'Expose more controls.',
      type: 'boolean',
      default: 'false',
    },
    callback: {
      description: 'Callback function to be invoked on timeout.',
      type: '() => void',
      default: 'undefined',
    },
  }}
/>

### Return Value

The return type depends on the `controls` option:

If `controls` is `false` (default):
