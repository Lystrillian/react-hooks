The `logicAnd` function (also available as `and` for convenience) performs a logical AND operation on multiple provided values or functions. It efficiently checks if all provided conditions evaluate to true.

## Usage

```tsx
import { logicAnd } from '#';
import * as React from 'react';

const Component = () => {
  const value1 = true;
  const value2 = false;
  const value3 = () => true;

  const result = logicAnd(value1, value2, value3);

  return (
    <div>
      <p>Logical AND result: {result.toString()}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { logicAnd } from '#';
import * as React from 'react';

const Component = () => {
  const [value1, setValue1] = React.useState(true);
  const [value2, setValue2] = React.useState(true);

  const result = logicAnd(value1, value2, () => true);

  return (
    <div>
      <p>Logical AND result: {result.toString()}</p>
      <button onClick={() => setValue1(false)}>Set value1 to false</button>
      <button onClick={() => setValue2(false)}>Set value2 to false</button>
    </div>
  );
};

export default Component;
```

## Type Declarations

```typescript
export declare function logicAnd(...args: Resolvable<any>[]): boolean;

export declare type Resolvable<T> = T | (() => T);
```

## Reference

#### logicAnd

| Property | Type              | Default | Description                          |
| -------- | ----------------- | ------- | ------------------------------------ |
| `args`   | `Resolvable<any>` | -       | The values or functions to evaluate. |

#### Resolvable

| Property | Type               | Description                                            |
| :------- | :----------------- | ------------------------------------------------------ |
| `T`      | `T   \| (() => T)` | A value or a function that returns a value of type`T`. |

### Returns

| Type      | Description                                                                                                                                                                                  |
| :-------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `boolean` | Returns `true` if all of the evaluated values are true; otherwise, `false`. If any of the provided values or functions are functions, they are called and their return values are evaluated. |
