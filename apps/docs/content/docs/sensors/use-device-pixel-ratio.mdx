## Installation

```bash
npx changeelog/reactuse@latest add useDevicePixelRatio
```

## Usage

```tsx
import { useDevicePixelRatio } from '#';

const Component = () => {
  const { pixelRatio } = useDevicePixelRatio();

  return (
    <div>
      <p>Device Pixel Ratio: {pixelRatio}</p>
    </div>
  );
};
```

## Reference

### useDevicePixelRatio

`useDevicePixelRatio` is a hook that returns the current device pixel ratio and updates when it changes.
