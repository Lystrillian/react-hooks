## Installation

```bash
npx changeelog/reactuse@latest add useTyping
```

## Usage

```tsx
import { useTyping } from '#';

const Component = () => {
  const isTyping = useTyping();

  return (
    <div>
      <p>User is typing: {isTyping ? 'Yes' : 'No'}</p>
      <input type="text" placeholder="Type here..." />
    </div>
  );
};
```

## Reference

### useTyping

`useTyping` is a hook that listens for `keydown` and `keyup` events on the document object to determine if the user is currently typing.

Returns a boolean value:
