## Installation

```bash
npx changeelog/reactuse@latest add useMousePosition
```

## Usage

```tsx
import { useMousePosition } from '#';

const Component = () => {
  const { x, y } = useMousePosition();

  return (
    <div>
      <p>
        Mouse position - X: {x}, Y: {y}
      </p>
    </div>
  );
};
```

## Reference

### useMousePosition

`useMousePosition` is a hook that listens for 'mousemove' events on the window object to capture and update the current mouse position. 

### Return Value

Returns an object with the following properties:
