`usePrecision` is a hook that formats a number to a specified precision.

## Usage

```tsx
import { usePrecision } from '#';
import * as React from 'react';

const Component = () => {
  const value = 3.14159;
  const precision = 2;

  const formattedValue = usePrecision({ precision, value });

  return (
    <div>
      <p>Original value: {value}</p>
      <p>Formatted value: {formattedValue}</p>
    </div>
  );
};

export default Component;
```

```tsx
import { usePrecision } from '#';
import * as React from 'react';

const Component = () => {
  const [value, setValue] = React.useState(1.23456);
  const [precision, setPrecision] = React.useState(3);

  const formattedValue = usePrecision({ precision, value });

  return (
    <div>
      <p>Original value: {value}</p>
      <p>Formatted value: {formattedValue}</p>
      <button onClick={() => setValue(4.56789)}>Update value</button>
      <button onClick={() => setPrecision(1)}>Update precision</button>
    </div>
  );
};

export default Component;
```

## Type Declarations

```typescript
export declare function usePrecision(
  options: PrecisionOptions,
): UsePrecisionReturn;

export declare type PrecisionOptions = {
  precision: number;
  value: number;
};

export declare type FormattedValue = number;

export declare type UsePrecisionReturn = FormattedValue;
```

## Reference

#### usePrecision

| Property  | Type               | Default | Description                                             |
| :-------- | :----------------- | :------ | :------------------------------------------------------ |
| `options` | `PrecisionOptions` | -       | An object containing the precision and value to format. |

#### PrecisionOptions

| Property    | Type     | Default | Description                                         |
| :---------- | :------- | :------ | :-------------------------------------------------- |
| `precision` | `number` | -       | The number of decimal places to round the value to. |
| `value`     | `number` | -       | The number to format.                               |

#### FormattedValue

| Type     | Description                                                                |
| :------- | :------------------------------------------------------------------------- |
| `number` | The formatted value, which is a number rounded to the specified precision. |

### Returns

| Type     | Description                                                                |
| :------- | :------------------------------------------------------------------------- |
| `number` | The formatted value, which is a number rounded to the specified precision. |
