## Installation

```bash
npx changeelog/reactuse@latest add useInterpolation
```

## Usage

```tsx
import { useInterpolation } from '#';

const Component = () => {
  const interpolate = useInterpolation();
  const result = interpolate(0, 100, 0.5);

  return (
    <div>
      <p>Interpolated value: {result}</p>
    </div>
  );
};
```

## Reference

### useInterpolation

`useInterpolation` is a hook that returns a memoized function for performing linear interpolation between two numbers.
