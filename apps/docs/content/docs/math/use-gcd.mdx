## Installation

```bash
npx changeelog/reactuse@latest add useGCD
```

## Usage

```tsx
import { useGCD } from '#';

const Component = () => {
  const calculateGCD = useGCD();
  const result = calculateGCD(48, 18);

  return (
    <div>
      <p>GCD of 48 and 18: {result !== null ? result : 'Error'}</p>
    </div>
  );
};
```

## Reference

### useGCD

`useGCD` is a hook that returns a memoized function for calculating the Greatest Common Divisor of two integers.
