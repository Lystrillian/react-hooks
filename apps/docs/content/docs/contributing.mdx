## Workflow

1. Create a new branch for your task: `git checkout -b feature/your-feature-name`
2. Make the necessary changes
3. Make sure all tests pass: `pnpm test`
4. Commit the changes: `git commit -am 'Add some feature'`
5. Send changes to your fork: `git push origin feature/your-feature-name`
6. Create a Pull Request
