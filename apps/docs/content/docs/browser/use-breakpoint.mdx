## Installation

```bash
npx changeelog/reactuse@latest add useBreakpoint
````

## Usage

```tsx
import { useBreakpoint } from '#';

const Component = () => {
  const { windowWidth, isBreakpointCrossed } = useBreakpoint(768);

  return (
    <div>
      <p>Window width: {windowWidth}px</p>
      <p>Is desktop: {isBreakpointCrossed ? 'Yes' : 'No'}</p>
    </div>
  );
};
```

## Reference

### useBreakpoint

`useBreakpoint` is a hook that tracks the window width and checks if it has crossed a specified breakpoint.

Returns a `UseBreakpointResult` object with the following properties:
