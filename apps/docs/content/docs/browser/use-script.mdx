## Installation

```bash
npx changeelog/reactuse@latest add useScript
```

## Usage

### Basic Usage

```tsx
import { useScript } from '#';

const Component = () => {
  useScript('https://example.com/external-script.js');

  return <div>Component with external script loaded</div>;
};
```

## Reference

### useScript

`useScript` is a hook that Dynamically loads an external script into your React component.

### Return Value

This hook does not return any value. It only handles the side effect of loading the script.
