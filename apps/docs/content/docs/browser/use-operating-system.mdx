## Installation

```bash
npx changeelog/reactuse@latest add useOperatingSystem
```

## Usage

```tsx
import { useOperatingSystem } from '#';
import * as React from 'react';

const Component = () => {
  const os = useOperatingSystem();

  return (
    <div>
      <p>Your operating system is: {os}</p>
    </div>
  );
};

export default Component;
```

## Reference

### useOperatingSystem

`useOperatingSystem` is a hook that detects the user's operating system based on the browser's user agent string.
