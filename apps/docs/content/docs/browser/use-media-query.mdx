## Installation

```bash
npx changeelog/reactuse@latest add useMediaQuery
```

## Usage

```tsx
import { useMediaQuery } from '#';

const Component = () => {
  const isWideScreen = useMediaQuery('(min-width: 1200px)');

  return (
    <div>{isWideScreen ? <WideScreenLayout /> : <NarrowScreenLayout />}</div>
  );
};
```

## Reference

### useMediaQuery

`useMediaQuery` is a hook that detects if a media query matches the current environment.

## Return Value

Returns a `boolean | undefined` value:

- `true` if the media query matches
- `false` if the media query doesn't match
- `undefined` if the match status is not yet determined (e.g., during server-side rendering)
