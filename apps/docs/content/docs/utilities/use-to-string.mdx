## Installation

```bash
npx changeelog/reactuse@latest add useToString
```

## Usage

```tsx
import { useToString } from '#';

const Component = () => {
  const staticString = useToString(123);
  const dynamicString = useToString(() => new Date().toLocaleString());

  return (
    <div>
      <p>Static value: {staticString}</p>
      <p>Dynamic value: {dynamicString}</p>
    </div>
  );
};
```

## Reference

### useToString

`useToString` is a hook that converts a value of any type to a string.
