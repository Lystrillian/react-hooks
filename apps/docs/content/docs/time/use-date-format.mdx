## Installation

```bash
npx changeelog/reactuse@latest add useDateFormat
```

## Usage

```
import { useDateFormat } from '#';

const Component = () => {
  const date = new Date();
  const formattedDate = useDateFormat(date, 'YYYY-MM-DD HH:mm:ss', {
    locales: 'en-US',
    customMeridiem: (hours) => (hours >= 12 ? 'PM' : 'AM'),
  });

  return <div>Formatted Date: {formattedDate}</div>;
};
```

## Reference

### useDateFormat

`useDateFormat` is a hook that formats a date according to a specified format string and options.

DateFormatOptions
An interface defining optional configuration for date formatting:

<TypeTable
  type={{
    locales: {
      description: 'The locale(s) to use for formatting.',
      type: 'string | string[]',
      default: 'undefined',
    },
    customMeridiem: {
      description: 'Custom function to format AM/PM indicator.',
      type: '(hours: number, minutes: number, isLowercase?: boolean, hasPeriod?: boolean) => string',
      default: 'undefined',
    },
  }}
/>

### Return Value

Returns a `string` representing the formatted date.
