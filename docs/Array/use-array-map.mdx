Array.map.

## Usage

```tsx
import { useArrayMap } from '#';

const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = useArrayMap(numbers, num => num * 2);
// doubledNumbers: [2, 4, 6, 8, 10]
```

Use with a custom callback function:

```tsx
import { useArrayMap } from '#';

const users = [
  { id: 1, name: 'John' },
  { id: 2, name: 'Jane' },
  { id: 3, name: 'Bob' },
];

const userNames = useArrayMap(users, user => user.name);
// userNames: ['John', 'Jane', 'Bob']
```
