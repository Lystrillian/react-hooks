# useSorted

<HookInfo category="Array" exportSize="Might vary, will be calculated on production" lastChanged="Will be updated based on git history" />

Sort an array using a custom compare function or options.

## Usage

```tsx
import { useSorted } from '#';

const numbers = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5];
const sortedNumbers = useSorted(numbers);
// sortedNumbers: [1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]
```

Use with a custom compare function:

```tsx
import { useSorted } from 'react-hook-port';

const users = [
  { id: 1, name: 'John' },
  { id: 2, name: 'Jane' },
  { id: 3, name: 'Alice' },
  { id: 4, name: 'Bob' },
];

const sortedUsers = useSorted(users, (a, b) => a.name.localeCompare(b.name));
// sortedUsers: [
//   { id: 3, name: 'Alice' },
//   { id: 4, name: 'Bob' },
//   { id: 2, name: 'Jane' },
//   { id: 1, name: 'John' },
// ]
```

Use with options:

```tsx
import { useSorted } from '#';

const numbers = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5];
const sortedNumbers = useSorted(numbers, {
  compareFn: (a, b) => b - a,
  dirty: true,
});
// sortedNumbers: [9, 6, 5, 5, 5, 4, 3, 3, 2, 1, 1]
// numbers: [9, 6, 5, 5, 5, 4, 3, 3, 2, 1, 1]
```
