Array.every.

## Usage

```tsx
import { useArrayEvery } from '#';

const list = [2, 4, 6, 8, 10];
const isEven = useArrayEvery(list, num => num % 2 === 0);
// isEven: true
```

Use with a predicate function:

```tsx
import { useArrayEvery } from '#';

const users = [
  { id: 1, active: true },
  { id: 2, active: true },
  { id: 3, active: false },
];

const allActive = useArrayEvery(users, user => user.active);
// allActive: false
```
