# useArrayDifference

<HookInfo category="Array" exportSize="Might vary, will be calculated on production" lastChanged="Will be updated based on git history" />

Returns a new array containing elements from list that are not present in values.

## Usage

```tsx
import { useArrayDifference } from '#';

const list = [0, 1, 2, 3, 4, 5];
const values = [4, 5, 6];
const result = useArrayDifference(list, values);
// result: [0, 1, 2, 3]
```

Use with a key or comparison function:

```tsx
import { useArrayDifference } from '#';

const list = [{ id: 1 }, { id: 2 }, { id: 3 }, { id: 4 }, { id: 5 }];
const values = [{ id: 4 }, { id: 5 }, { id: 6 }];

const result1 = useArrayDifference(list, values, 'id');
// result1: [{ id: 1 }, { id: 2 }, { id: 3 }]

const result2 = useArrayDifference(
  list,
  values,
  (item, othItem) => item.id === othItem.id
);
// result2: [{ id: 1 }, { id: 2 }, { id: 3 }]
```
