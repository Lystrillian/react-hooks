Array.find.

## Usage

```tsx
import { useArrayFind } from '#';

const numbers = [1, 2, 3, 4, 5];
const firstEvenNumber = useArrayFind(numbers, num => num % 2 === 0);
// firstEvenNumber: 2
```

Use with a custom predicate function:

```tsx
import { useArrayFind } from '#';

const users = [
  { id: 1, name: 'John', active: false },
  { id: 2, name: 'Jane', active: true },
  { id: 3, name: 'Bob', active: false },
];

const firstActiveUser = useArrayFind(users, user => user.active);
// firstActiveUser: { id: 2, name: 'Jane', active: true }
```
