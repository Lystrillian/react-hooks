Array.findIndex hook for React.

## Usage

Use with an array of multiple refs:

```tsx
import { useArrayFindIndex } from '#';

const MyComponent = () => {
  const users = [
    { id: 1, name: 'Alice' },
    { id: 2, name: 'Bob' },
  ];

  const findUserIndexById = user => user.id === 2;
  const userIndex = useArrayFindIndex(users, findUserIndexById);

  return <div>User Index: {userIndex}</div>; // Output: 1
};
```
