# useArrayUnique

<HookInfo category="Array" exportSize="Might vary, will be calculated on production" lastChanged="Will be updated based on git history" />

Unique array of items.

## Usage

```tsx
import { useArrayUnique } from '#';

const items = [1, 2, 2, 3, 4, 4, 5];
const uniqueItems = useArrayUnique(items);
// uniqueItems: [1, 2, 3, 4, 5]
```

Use with a custom compare function:

```tsx
import { useArrayUnique } from '#';

const items = [
  { id: 1, name: 'John' },
  { id: 2, name: 'Jane' },
  { id: 2, name: 'Jane' },
  { id: 3, name: 'Alice' },
];

const uniqueItems = useArrayUnique(items, (a, b) => a.id === b.id);
// uniqueItems: [
//   { id: 1, name: 'John' },
//   { id: 2, name: 'Jane' },
//   { id: 3, name: 'Alice' },
// ]
```

## Type Declarations

